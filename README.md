Brand design portfolio site for Studiotra | Petra Hwang.

## Deploy Structure

```
/
├── index.html          # Home
├── about.html          # About
├── contact.html        # Contact
├── work.html           # Work listing
├── mail.php            # Contact form handler
├── assets/             # CSS, JS, images, vendors
├── img/                # Logo, favicon, shared images
├── work/               # Project detail pages
│   ├── pe-re-investor-deck.html
│   ├── amazonads-fizz.html
│   ├── alphi-presentation.html
│   ├── td-financial-literacy.html
│   ├── td-summer-reading.html
│   └── timvo-work.html
├── docs/               # Planning & reference (exclude from deploy)
└── ASSETS-GUIDE.md     # Image asset documentation
```

## Deploy

Deploy the root folder. Exclude `docs/` from production if desired.
